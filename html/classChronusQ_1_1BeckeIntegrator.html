Detailed Description

template<class _QTyp1>
class ChronusQ::BeckeIntegrator< _QTyp1 >

Becke Molecular Integrator.

Integrates a molecular integrand using the Becke sphere integration scheme. Specializes SphereIntegrator

See J. Chem. Phys. 88, 2547 (1988)
