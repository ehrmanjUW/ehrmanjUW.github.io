Detailed Description

template<typename MatsT>
class ChronusQ::Quantum< MatsT >

The Quantum class. The typed abstract interface for all classes which can define a 1PDM and compute 1 and 2 body properties.

Adds knowledge of storage type and general property evaluation schemes to QuantumBase.
