Detailed Description

template<class _QTyp1>
class ChronusQ::SphereIntegrator< _QTyp1 >

Sphere Integrator.

Handles the integration of a 3D (cartesian) function over a ball of a specified radius.

Templated over radial quadrature scheme, angular quadrature taken to be Lebedev. Specializes Integrator2D.
